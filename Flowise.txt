Flowise:
--------

	It's an open source tool that makes it easy to prototype langchain application using an easy to use user interface.

	It allows us to create AI applications using a very simple drag and drop user interface and can be a usefull tool for prototyping langchain application


setup:
------

	## Pre-requisite: ensure NodeJS is installed on machine. Node v18.15.0 or v20 and above is supported. ##

  1. Install Flowise:
	
	npm install -g flowise

  2. Start Flowise:

	npx flowise start

  3. Open: http://localhost:3000
